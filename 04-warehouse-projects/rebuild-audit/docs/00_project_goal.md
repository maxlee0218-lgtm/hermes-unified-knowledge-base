# 项目目标

当前项目目标是重构并存库 `ADS_SC_XL_13` 相关数仓链路。

这次工作的目标不是：

- 不是只修一个最终表
- 不是只看 `combined_002` 壳层
- 不是只做最终结果对账

真正目标是把下面这些链条沉淀成一个可复现、可追踪、可持续推进的工程：

- 主链
- 子链
- 支撑链
- 字典链
- 日期骨架
- 人工确认链
- 入库量链

当前优先验收目标是：

1. `ads_sc_xl_13_defined_manuf_line_name_combined_001`
2. `ads_sc_xl_13_defined_manuf_line_name_combined_002`

当前工程目标分三层：

1. 先把链路画清楚
2. 再把 SQL、依赖、缺口沉淀成稳定文件
3. 最后让 ChatGPT 能基于这个仓库继续推进审计、重构、对账和验收
