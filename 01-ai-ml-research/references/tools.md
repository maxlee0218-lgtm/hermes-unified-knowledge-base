# 工具清单

> 基于 Karpathy LLM Wiki 最佳实践

## 必装工具

| 工具 | 作用 | 安装 |
|------|------|------|
| Obsidian | 知识库IDE | 官网下载 |
| ripgrep (rg) | 全文搜索 | `apt install ripgrep` |
| Git | 版本控制 | 已安装 |

## 推荐工具

| 工具 | 作用 | 安装 |
|------|------|------|
| Obsidian Web Clipper | 网页转Markdown | 浏览器插件 |
| marker-pdf | PDF转Markdown | `pip install marker-pdf` |
| Pandoc | 多格式导出 | `apt install pandoc` |
| fzf | 模糊搜索 | `apt install fzf` |

## 可选工具

| 工具 | 作用 | 适用场景 |
|------|------|----------|
| qmd | 语义搜索 | 知识库>100篇 |
| Ollama | 本地Embedding | 需要8G+内存 |
| Zotero | 论文管理 | 学术场景 |
| Readwise | 标注同步 | 阅读场景 |
