## 6. 报表 SQL 解读

```sql
select * from (
    -- 历史数据：取非当天的所有数据
    select *,1 from ads_gx_xs_20_03
    where data_date != CURRENT_DATE 
      and (qx_a_weight is not null or qx_b_weight is not null or qx_c_weight is not null or ajwjl is not null)
    union all
    -- 当天数据：取当天的数据
    select *,2 from ads_gx_xs_04_03
    where data_date = CURRENT_DATE 
      and (qx_a_weight is not null or qx_c_weight is not null or ajwjl is not null)
) a
order by attr1, surface_thickness, surface_number
```

### 逻辑说明

1. **ads_gx_xs_20_03**：取历史数据（排除当天）
   - 标记：1（历史数据标识）
   - 过滤：qx_a/b/c_weight 或 ajwjl 不为空

2. **ads_gx_xs_04_03**：取当天数据
   - 标记：2（当天数据标识）
   - 过滤：同上

3. **UNION ALL**：合并历史+当天

4. **ORDER BY**：按 attr1, surface_thickness, surface_number 排序

## 7. 发现的问题

1. **attr1 历史数据缺失**：ads_gx_xs_20_03 的 attr1 全部为 NULL，04_03 有值
2. **空值率高**：ajwjl 空值率 60%~78%，qx_c_weight 空值率 36%~48%
3. **数据量差异大**：历史表 17,145 条，当天表仅 76 条
4. **无主键**：两张表都没有主键或唯一索引

## 8. 建议

1. 检查 attr1 历史数据缺失原因
2. 评估空值字段的业务影响
3. 考虑为两张表添加主键
4. 确认 20_03 和 04_03 的数据更新机制
