# 大模型幻觉（LLM Hallucination）

## 定义

大语言模型在生成回复时，输出看似合理但实际上不真实、不准确或无法验证的内容。幻觉源于训练数据缺失/错误/过时、参数拟合偏差、推理逻辑断层、无实时事实校验能力、生成算法优先追求流畅性而非真实性。

## 7 大类幻觉

| 类型 | 定义 | 典型表现 |
|------|------|---------|
| **事实性幻觉** | 捏造事实、虚假数据、编造来源 | "据了解、相关数据显示" |
| **逻辑性幻觉** | 推理断层、因果倒置、循环论证 | 无前提的结论、逻辑跳跃 |
| **知识性幻觉** | 知识盲区、过时知识、领域偏差 | 用通识替代专业标准 |
| **表达性幻觉** | 过度引申、偷换概念、模糊边界 | 夸大效果、超范围绝对化 |
| **长上下文幻觉** | 上下文遗忘、信息错位、跨轮次矛盾 | 前后说法不一致 |
| **迎合性幻觉** | 迎合用户预期、顺着引导性提问编造 | 为了让用户满意而说谎 |
| **多模态幻觉** | 图文不符、音文错位、模态信息编造 | 编造图片中不存在的内容 |

## 核心原则

**真实性优先于流畅性、优先于用户预期、优先于回答完整性。**

严禁为了完成回答、迎合用户、规避"不知道"的表述而编造、推测、模糊化输出虚假信息。
