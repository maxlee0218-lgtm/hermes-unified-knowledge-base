## 一、安装 DataGrip

### 1.1 下载地址（推荐）
- **JetBrains 中国官网**：https://www.jetbrains.com.cn/datagrip/
- 或 JetBrains Toolbox：https://www.jetbrains.com.cn/toolbox-app/

### 1.2 安装步骤
1. 访问 JetBrains 中国官网
2. 点击"下载"按钮
3. 运行安装程序
4. 选择安装路径（建议 D:/Program Files/JetBrains/DataGrip）
5. 完成安装

## 三、常用操作

### 3.1 查看表结构
```sql
DESCRIBE dwd_tms_delivery_que_history;
```

### 3.2 查看数据量
```sql
SELECT COUNT(*) FROM dwd_tms_delivery_que_history;
```

### 3.3 查看最近数据
```sql
SELECT * FROM dwd_tms_delivery_que_history 
ORDER BY data_date DESC 
LIMIT 10;
```

## 五、注意事项

1. **只读操作**：默认只执行 SELECT/SHOW
2. **不写库**：禁止 INSERT/UPDATE/DELETE
3. **不删表**：禁止 DROP/TRUNCATE
4. **密码安全**：不在公共场合展示密码
