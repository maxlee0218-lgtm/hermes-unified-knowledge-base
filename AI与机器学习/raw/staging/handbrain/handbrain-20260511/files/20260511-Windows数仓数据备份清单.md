## 六、备份步骤建议

### 6.1 立即执行
1. **复制核心项目**：
   ```powershell
   # 创建备份目录
   mkdir D:/backup-reinstall
   
   # 复制dbt项目
   robocopy D:/data-warehouse/01-dbt-duckdb D:/backup-reinstall/01-dbt-duckdb /E
   
   # 复制DataX配置
   robocopy D:/data-warehouse/02-datax D:/backup-reinstall/02-datax /E
   
   # 复制监控项目
   robocopy D:/data-warehouse/03-monitoring D:/backup-reinstall/03-monitoring /E
   
   # 复制报告
   robocopy D:/AIWorker/reports D:/backup-reinstall/reports /E
   ```

2. **导出数据库**：
   ```powershell
   # DuckDB数据库
   copy D:/data-warehouse/01-dbt-duckdb/dw_rebuild.duckdb D:/backup-reinstall/
   ```

3. **记录配置**：
   - 截图数据库连接配置
   - 记录环境变量
   - 记录定时任务

### 6.2 验证备份
1. 检查备份目录大小
2. 抽查关键文件是否存在
3. 确认配置文件完整
