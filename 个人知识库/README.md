# Obsidian 个人知识库

## 📖 简介
本目录是个人知识管理的主要区域，包含日常笔记、学习记录、思考总结等内容。

## 🗂️ 内容分类

### 日常笔记
- 工作日志
- 会议记录
- 任务清单
- 项目笔记

### 学习记录
- 技术学习笔记
- 读书笔记
- 课程笔记
- 研究笔记

### 个人管理
- 目标规划
- 时间管理
- 思考总结
- 创意灵感

### 知识网络
- 概念关系图
- 思维导图
- 知识链接
- 标签系统
